# UI Copy Guide — NCTR Platforms

> **Rule**: Never use *crypto, blockchain, token(s), wallet, staking, yield, portfolio, Web3, investing, liquidity, bounty*.
> **Preferred**: rewards, status, earn, participate, contribute, access, commit, NCTR.
> **Core message**: "Don't Buy It. Earn It. Own It."

## 4 · Copywriting Principles

1. **Lead with the benefit**, not the mechanism. "Earn NCTR" not "Receive token allocation."
2. **Use active voice**. "You earned 50 NCTR" not "50 NCTR has been credited."
3. **Keep it human**. Short sentences. Conversational tone. No jargon.
4. **Status-first framing**. Every action should feel like building toward something — Crescendo status, better rewards, community standing.
5. **Celebrate progress**. Use 🎉 and positive language for milestones.
6. **Be specific**. "Earn up to 8% back" beats "Earn rewards."
